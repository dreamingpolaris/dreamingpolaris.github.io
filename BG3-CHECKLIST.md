# Checklist for Baldur's Gate 3

This assumes a "good" path run, mainly because I mostly play "good" runs.
Also that this is a Tactician/Honour Mode run, where pathing matters slightly more.

 - [ ] Prologue/Nautiloid
   - [ ] Collect the Cartilaginous Chests
     - [ ] Nursery
       - [ ] L
       - [ ] R
     - [ ] Ceremorphosis Chamber
   - [ ] Free Us
   - [ ] Encounter Lae'zel/Imp Fight
   - [ ] Free Shadowheart
   - [ ] Block off the entrance to the Helm with the chests
   - [ ] Helm
     - [ ] Get the Everburn Blade
     - [ ] Transponder
 - [ ] Act One/Wilderness
   - [ ] Ravaged Beach
     - [ ] Recruit Shadowheart
     - [ ] Recruit Astarion
     - [ ] Recruit Gale
     - [ ] Recruit Lae'zel (now or later)
     - [ ] Intellect Devourers
     - [ ] Kill the dying mind flayer
   - [ ] Overgrown Ruins
     - [ ] Get inside of the Ruins
       - [ ] Dealing with Gimblebock
         - [ ] Then Andorn
       - [ ] Locked hatch (DC 20 Sleight of Hand)
     - [ ] Talk to Jergal
   - [ ] Emerald Grove
     - [ ] Fight at the Gate
     - [ ] Convince Rolan to Stay
     - [ ] Recruit Wyll
     - [ ] Convince Arka to stand down (Gale point)
     - [ ] Save Findal in the Underground Passage
       - [ ] Get Nature's Snare (a surprise tool that will help us Soon)
     - [ ] Respec ALL of your companions to their desired builds
       - [ ] (Optional) Use Wyll to trade with Arron, Ethel, and Dammon to farm STR pots, scrolls, and +1 basic weapons.
         - [ ] Use the Bind Hexed Weapon bug to do this on Nature's Snare
         - [ ] Note that traders refresh on every level up/respec, so respecing all of Tav/Durge, Lae'zel, Shadowheart, Astarion, and Gale can get you 5 * 3 * 3 = 30 STR pots, more than enough for the rest of Acts 1 & 2
 - [ ] Act Two/Shadow Cursed Lands
 - [ ] Act Three/Baldur's Gate
